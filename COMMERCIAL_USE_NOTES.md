# Commercial Use Notes

GX1 Video Forge is approved as a private MVP only. Do not activate a public domain for customers until security review, asset review, provider terms review, and manual QA are complete.

## Safe boundaries

- Keep the original MIT license and attribution intact.
- Configure `GX1_ACCESS_TOKEN` before any internet-accessible deployment.
- Do not commit secrets, API keys, provider tokens, or customer data.
- Use only commercially cleared stock footage, audio, fonts, voices, and scripts for paid distribution.
- Keep autoposting disabled unless a later authorized integration explicitly adds platform-specific controls.
- Treat generated content as requiring human review before publication.

## Not yet included

This MVP does not include Hotmart integration, paid SaaS login, customer billing, public multi-tenant isolation, or platform autoposting.
