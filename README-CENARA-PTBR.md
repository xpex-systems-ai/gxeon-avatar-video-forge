# Cenara — MVP privado GXEON

Cenara é um MVP privado da GXEON para criação de vídeos com IA voltados a criativos, anúncios e conteúdo de venda.

> Powered by GXEON

## Base e atribuição

Cenara é baseado no projeto open source MoneyPrinterTurbo, criado por Harry, e preserva a licença MIT original. Este repositório mantém a atribuição upstream e não reivindica autoria do projeto original.

## Status do produto

- MVP privado, ainda não é um SaaS público.
- Acesso operacional protegido por `GX1_ACCESS_TOKEN` em produção/modo privado.
- O nome público do produto é Cenara; GXEON aparece como proprietário/assinatura e em documentação de compliance.

## Escopo atual

O MVP atual cobre roteiro, voz, legenda, cenas, materiais de vídeo e geração/renderização de vídeos. A camada de avatar é um módulo futuro e não faz parte do produto atual pronto para venda pública.

## Público-alvo

Gestores de tráfego, agências, afiliados, donos de loja e infoprodutores que precisam criar vídeos para aquisição, anúncios e páginas de venda.
