# GX1 Safe Asset Mode

GX1 Video Forge retains upstream sample/background music and media directories for compatibility, but those assets are not commercially cleared by default.

## Rules

1. Do not use default YouTube-derived or bundled music commercially until every file has documented source, license, and permitted-use evidence.
2. Prefer self-owned, commissioned, royalty-free, or provider-licensed assets with records stored outside the repository.
3. For client, ad, Hotmart, or public videos, keep an asset log containing file name, source URL/vendor, license type, date acquired, and allowed platforms.
4. Do not delete upstream assets without explicit authorization; flag them as unsafe for commercial use instead.
5. Human review is required before publishing any generated video.
